# CONTRIBUTING.md

Thank you for your interest in contributing.

This organization provides small, self-contained **training repositories**
that pair a model architecture with a corpus.

## Adding a New Training Repository or Corpus

To contribute:

- a new corpus, or
- a new training repository using an existing architecture,

please review the documentation and standards in the relevant associated repository.

## Participating

- Follow the project Code of Conduct.
- Keep contributions focused on clarity and shared value.

If you are unsure where a contribution belongs,
open an issue in this organization or a
related repository to discuss before beginning.
